Информация о программе Rohos Logon Key

1. Производитель и история разработки

  • Разработчик: Tesline-Service S.R.L. (Молдова)
  • Первая версия: 2006 год
  • Актуальная версия: 7.0 (2023)
  • Модель распространения: Freemium

3. Основные возможности

  • Двухфакторная аутентификация
  • Вход в Windows без пароля
  • Шифрование USB-накопителей (AES-256)
  • Поддержка YubiKey и мобильного приложения
  • Интеграция с Active Directory
  • Резервное восстановление доступа

4. Особенности

  • Меню: Управление через системный трей
  • Навигация: Мастер настройки
  • Интересные функции:
    • Виртуальные USB-ключи
    • Режим «Экран блокировки» с геолокацией
    • Генерация OTP-паролей
    • Совместимость с Windows Hello
